 Ejercicios de NumPy - Aprendizaje Automático

Este repositorio contiene ejercicios prácticos de la materia **Aprendizaje Automático**, realizados en Python utilizando la librería NumPy.  
El objetivo es fortalecer la comprensión de operaciones numéricas, manipulación de arrays y matrices, y aplicar estos conceptos en el contexto de la ciencia de datos.

---

## 📂 Contenido
- **Semana1/Ejercicios_Num.py**  
  Ejercicios iniciales que incluyen:
  - Creación de arrays con `arange` y `random`
  - Operaciones básicas de suma y multiplicación
  - Generación de matrices aleatorias
  - Cálculo de estadísticas simples (máximo, mínimo, promedio)
  - Extracción de diagonales y sumas de filas/columnas

---
